# CBA-Task-16
How to run the project:
Clone the project using
 git clone " https://github.com/Sathvika2023/CBA-Task-16.git"
 
1. Install node js or check for version
   node -v
2. change the directory
    cd my-react-task16
3. Install node modules
    npm install
4. Start the application
    npm run dev
